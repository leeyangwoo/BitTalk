GITHUB 규칙

-- Remote(Origin)에서 최신 코드 받아오기
git pull origin master


-- 로컬에서 작업시 브랜치 따기
git branch (브랜치이름)
git checkout (브랜치이름) -->브랜치이동/ git shell에서 브랜치 이름 바뀌었는지 확인


-- 로컬에서 작업하면서 자기 브랜치에서 commit하기
1.git add .
2.git commit -m "작업 내용"
3.git push origin (브랜치이름)
